# company-blog
Modified code along project from Jose Portilla's Flask Course on Udemy

To run this code with [Anaconda](https://www.anaconda.com/):
1) [Clone](https://docs.github.com/en/github/creating-cloning-and-archiving-repositories/cloning-a-repository) the repository.
2) Run 'conda env create'.
3) Run 'source activate company_blog'.
